# Política de Privacidade

**Última atualização:** junho de 2026

## 1. Introdução

Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações dos usuários que utilizam nosso Bot no Discord.

## 2. Dados Coletados

Coletamos apenas os dados estritamente necessários para o funcionamento do serviço:

- **ID do usuário no Discord** — para identificação e gerenciamento da conta
- **Identificadores fornecidos pelo usuário** — necessários para a entrega das funcionalidades solicitadas
- **Data e hora de cadastro** — para controle interno do serviço

Não coletamos senhas, tokens, dados bancários ou qualquer informação sensível.

## 3. Uso dos Dados

Os dados coletados são utilizados exclusivamente para:
- Identificar o usuário dentro do sistema
- Entregar as funcionalidades contratadas
- Controle de acesso e gerenciamento interno

## 4. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados com terceiros, exceto quando exigido por lei.

## 5. Armazenamento e Segurança

Os dados são armazenados de forma segura e acessados apenas pelos administradores do serviço. Adotamos medidas razoáveis para proteger as informações contra acesso não autorizado.

## 6. Retenção de Dados

Os dados são mantidos enquanto o usuário estiver ativo no sistema. Após o encerramento do serviço ou solicitação do usuário, os dados serão removidos.

## 7. Direitos do Usuário

O usuário tem direito a:
- Solicitar a visualização dos seus dados armazenados
- Solicitar a exclusão dos seus dados
- Revogar o uso do serviço a qualquer momento

Para exercer esses direitos, entre em contato pelo servidor oficial no Discord.

## 8. Cookies e Rastreamento

O Bot não utiliza cookies nem realiza rastreamento de atividades fora do Discord.

## 9. Alterações nesta Política

Podemos atualizar esta Política periodicamente. Recomendamos revisá-la regularmente.

## 10. Contato

Em caso de dúvidas sobre esta Política de Privacidade, entre em contato pelo servidor oficial no Discord.
